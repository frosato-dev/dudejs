# DudeJS


## How to use scripts

```sh
$ npm run dude:format
$ npm run dude:lint
```


```sh
$ yarn dude:format
$ yarn dude:lint
```


## Development commands

* Run tests: `$ npm run test`
* Format code: `$ npm run format`
* Lint code: `$ npm run lint`
* Build: `$ npm run build`
* Add contributor: `$ npm run contributors:add <github-username> <contribution-type>`
* Deploy: `$ npm run deploy`
